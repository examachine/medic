Pieces and Ties

MEDIC is composed of three parts: a base library which defines DICOM data structures, a kernel library which provides core DICOM functionality and a portability layer for filesystem, thread and socket support. The kernel library is further divided into two components: low level and high level kernel. In the low level fundamental data types and coder support reside, in the high level are SOP classes and DICOM services. Figure 1 depicts the high level composition.

High Level

The high level portion of Kernel interface features:

Class SOP: Abstract base class of all SOP Classes, this abstract class defines the minimal interface an SOP should support such as construction, destruction and encoding/decoding.

Class DException: Base class for all DICOM Exceptions. Exceptions are thrown by many mechanisms in the system, for instance when a protocol error occurs. The DException has many subclasses which indicate different kinds of errors. The MEDIC user can catch and assess these errors.

Figure 5: Exception Class Hierarchy

Class DStreamBuffer: A stream buffer which implements DStream interface. This buffer is appropriate for PDU/PDV style input/output.

Class DBinaryFileStream: A subclass of DStream which reads and writes to regular files.

Class DTCPStream: A subclass of DStream which handles client stream connections over TCP.

Class DServerStream: A special subclass of DTCPStream which is better suited for server connections.

Class DApplication: This is the abstraction of a DICOM Application Entity. It is responsible for managing various configuration parameters and maintaining system data. There is only one instance of DApplication called dicom_app in an application program.

Class StorageManager: A class which implements a simple minded virtual memory scheme for large objects.

Class DContext: The abstraction for a DICOM association, a subclass of DContext which manages execution of DICOM services.

Class DTCPContext: Subclass of DContext which supervises both client and server association, and handling of DIMSE-C and DIMSE-N services within that association.

Class DFileContext: Subclass of DContext responsible for reading and writing DICOM files. This class implements a tentative subset of DICOM File Services

Class DImageStacker: This class is declared if MEDIC_ARCHIVE symbol is defined. The class checks to see if a series is complete and if so sends a special message to the archive system.

Within the MEDIC Kernel, platform specific features are kept minimum and ANSI C++ library is used for all elementary data structures: list<T>, vector<T>, map<K,T> are used extensively.
